üíª System Requirements

ü™ü Windows

  • OS: Windows 10 (1903+) or Windows 11
  • Processor: Intel/AMD x64 or ARM64
  • Memory: 4 GB RAM minimum, 8 GB recommended
  • Storage: 500 MB available space
  • Additional: .NET Framework 4.7.2+

üçé macOS

  • OS: macOS 10.15 Catalina or later
  • Processor: Intel x64 or Apple Silicon (M1/M2)
  • Memory: 4 GB RAM minimum, 8 GB recommended
  • Storage: 500 MB available space
  • Additional: Xcode Command Line Tools

üêß Linux

  • OS: Ubuntu 18.04+, Fedora 32+, or equivalent
  • Processor: x64 processor
  • Memory: 4 GB RAM minimum, 8 GB recommended
  • Storage: 500 MB available space
  • Additional: GLIBC 2.28+, GTK 3.22+

ü§ñ AI Requirements

For optimal AI agent performance, we recommend:

  • Internet connection for cloud AI providers (OpenAI, Claude)
  • Ollama for local AI (optional but recommended)
  • 8 GB+ RAM for running multiple AI agents simultaneously
  • SSD storage for faster AI response times

ü™ü Windows Deployment

üì¶ Method 1: Installer (Recommended)

  1. Download Weezy-Setup.exe from the download page
  2. Right-click the installer and select "Run as administrator"
  3. Follow the installation wizard
  4. Launch Weezy from Start Menu or Desktop shortcut
PowerShell Installation 
# Download and install via PowerShell
Invoke-WebRequest -Uri "https://github.com/weezy-dev/weezy/releases/latest/download/Weezy-Setup.exe" -OutFile "WeezySetup.exe"
Start-Process -FilePath "WeezySetup.exe" -Verb RunAs

üìÅ Method 2: Portable Version

  1. Download Weezy-win32-x64.zip
  2. Extract to your preferred location (e.g., C:\Program Files\Weezy)
  3. Run Weezy.exe from the extracted folder
  4. Optionally, create desktop shortcut

üç´ Method 3: Chocolatey

Chocolatey Package Manager 
# Install via Chocolatey
choco install weezy

# Update to latest version
choco upgrade weezy

üçé macOS Deployment

üíø Method 1: DMG Installer (Recommended)

  1. Download Weezy-darwin-universal.dmg
  2. Double-click the DMG file to mount it
  3. Drag Weezy to your Applications folder
  4. Launch Weezy from Applications or Spotlight

üîê Security Notice

On first launch, macOS may show a security warning. To resolve:

  1. Go to System Preferences > Security & Privacy
  2. Click "Open Anyway" for Weezy
  3. Or run: sudo xattr -rd com.apple.quarantine /Applications/Weezy.app

üç∫ Method 2: Homebrew

Homebrew Installation 
# Add Weezy tap
brew tap weezy-dev/weezy

# Install Weezy
brew install weezy

# Update to latest version
brew upgrade weezy

üê≥ Docker Deployment

Run Weezy in a containerized environment for consistent deployment across any system.

Docker Run 
# Pull and run Weezy container
docker run -d \
  --name weezy \
  -p 8080:8080 \
  -v /path/to/your/projects:/workspace \
  -v weezy-data:/home/weezy/.weezy \
  weezy/weezy:latest

# Access Weezy at http://localhost:8080
Docker Compose 
# docker-compose.yml
version: '3.8'

services:
  weezy:
    image: weezy/weezy:latest
    container_name: weezy
    ports:
      - "8080:8080"
    volumes:
      - ./projects:/workspace
      - weezy-data:/home/weezy/.weezy
    environment:
      - WEEZY_AI_PROVIDERS=openai,claude,ollama
      - WEEZY_THEME=dark
    restart: unless-stopped

volumes:
  weezy-data:

üîß Troubleshooting

‚ùå Installation Fails

Problem: Installer shows error or fails to complete
Solutions:
  • Run installer as administrator (Windows)
  • Disable antivirus temporarily during installation
  • Clear temporary files and try again
  • Download fresh installer file

ü§ñ AI Agents Not Working

Problem: AI agents fail to respond or show errors
Solutions:
  • Check internet connection for cloud AI providers
  • Verify API keys in settings
  • Install Ollama for local AI: curl https://ollama.ai/install.sh | sh
  • Restart Weezy and try again

üêß Linux Display Issues

Problem: UI appears broken or fonts are missing
Solutions:
  • Install required fonts: sudo apt install fonts-inter
  • Update GTK libraries: sudo apt update && sudo apt upgrade
  • Try different display backend: weezy --disable-gpu
  • Check display scaling settings

‚ö° Performance Issues

Problem: Weezy runs slowly or uses too much memory
Solutions:
  • Close unused extensions and panels
  • Reduce number of concurrent AI agents
  • Increase system RAM if possible
  • Use local AI (Ollama) instead of cloud providers
